# 두수의 곱
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 * num2;
    }
}
```
# 두수의 차
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 - num2;
    }
}
```
# 두수의 합
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 + num2;
    }
}
```
