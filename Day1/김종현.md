# 두수의 곱
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 * num2;
    }
}
```

<br>

# 두수의 차
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 - num2;
    }
}
```

<br>

# 두수의 합
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 + num2;
    }
}
```

<br>

# 몫 구하기
```
class Solution {
    public int solution(int num1, int num2) {
        return num1 / num2;
    }
}
```
