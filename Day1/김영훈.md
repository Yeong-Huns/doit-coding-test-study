# a와 b 출력하기
```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        System.out.printf("a = %d%nb = %d",a,b);
    }
}
```

# 두 수의 합
```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 + num2;
    }
}
```
